JavaScript

Definition

JavaScript is a scripting language used to make web pages interactive, dynamic, and functional. It can update content, control multimedia, and handle user interactions.

Uses

  • Adding interactivity like buttons, forms, and alerts
  • Changing HTML & CSS dynamically
  • Validating form inputs
  • Creating animations and effects
  • Building full web applications

Types

  • Internal JavaScript → Written inside <script> tag on the same page
  • External JavaScript → Written in a separate .js file
  • Inline JavaScript → Written directly inside an HTML tag (not recommended)
